## Final Project: VIX Sentiment Multi-Asset Dashboard

**Title:** *Navigating Market Sentiment: A Multi-Asset Performance Study Based on the VIX Index*  
**Files:**  
- `Yutong_Wu_6191_Final.ipynb`  
- `Yutong_Wu_6191_Report.docx`  

This project analyzes how major asset classes behave across different market sentiment regimes (Low / Medium / High VIX), covering:

- Returns & volatility  
- Sharpe ratios  
- Drawdowns  
- Correlation shifts  
- Sentiment-tagged statistical distributions  
- VIX historical evolution  
- Regime-based asset allocation strategies  

A full interactive **Dash + Plotly dashboard** is included, allowing users to explore:

- Multi-asset metrics  
- VIX history  
- Heatmaps & violin plots  
- Strategy backtests  
- Asset filters, sentiment filters, and interactive controls
